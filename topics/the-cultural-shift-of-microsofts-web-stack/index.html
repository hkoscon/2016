The Cultural Shift of Microsoft’s Web Stack

By Bart Verkoeijen (bgever)

This talk covers how Microsoft is adopting an open approach to web development with the Microsoft stack

This talk covers how Microsoft is adopting an open approach to web development with the Microsoft stack, including hosting with Azure, open-source cross-platform .NET, support for other web platforms like Node.js, TypeScript, and even the Visual Studio Code editor. Learn about what benefits this brings to you as a developer, and how you can participate in this open development. A lot has happened in the last few years, you could even say a cultural shift is happening, and there’s a lot to be excited about with the new Microsoft.

Biography

Bart is a full-stack senior web developer at Quantifeed, where he builds the frontend for a FinTech web platform for Asia’s mass affluent to invest in market trends. His current technology stack includes Azure, ASP.NET, AngularJS and TypeScript

He is active in the community as co-organizer of the Microsoft Developers HK meetup, and organizer of the Friday Beer Bay meetup for Web, Tech, and Startups.
