Developing applications with React and Redux

By Anish Patil (Anish)

This talk will focus on how to use react and redux for development of web Apps.

Writing modular web components that can be used by other components seamlessly is arduous task. The talk would be focused on existing problems in web development i.e spaghetti code and maintaining state of applications that leads to hard to spot errors.
